# CoreInventory

CoreInventory is a modular Inventory Management System (IMS) built for the Odoo x Indus University Hackathon.

## Features
- Product Management
- Incoming Stock (Receipts)
- Outgoing Stock (Delivery Orders)
- Internal Transfers
- Stock Adjustment
- Dashboard Analytics

## Tech Stack
Frontend: React + Vite + TypeScript
Backend: Node.js + Express
Database: MySQL
